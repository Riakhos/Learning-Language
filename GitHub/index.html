Lignes de commande :

Les lignes de commande commençent toujours par l'outil Dans ce courts, l'outil est git.

git init

git add index.html ou git add . (pour tous les fichiers)

git commit -m "first commit"

git branch -M main

git remote add origin

git push -u origin main

Shortcuts

## avec 2 dièses, Titre Niveau 2

### avec 3 dièses, Titre Niveau 3

> avec 1 chevron, Citation niveau

** Entre 2 astérisques, Texte en gras **
